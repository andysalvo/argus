Argus

AI instruction file reviewer for GitHub.

Automatically reviews CLAUDE.md, AGENTS.md, .cursorrules, and 9 more file patterns. Scores against the Argus Governance Standard v1.1 -- 13 checks, document-type-aware scoring, zero false positives.

Try it locally

$ npx argus score CLAUDE.md
